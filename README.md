## Latest Update

This project was updated to include the following improvements:

- Added a consistent logo block on every HTML page (`index.html`, `about.html`, `gallery.html`, `menu.html`, `contact.html`).
- Added shared logo styling in `css/style.css` for `.logo` and `.logo-img`.
- Implemented menu cart functionality in `script/script.js`:
  - Add-to-cart button handlers
  - Cart item rendering
  - Cart count updates
  - Modal open/close behavior
  - Checkout alert and cart reset
- Added cart/modal styles in `css/style.css` for `.modal`, `.cart-item`, and `.checkout-btn`.

## How to use

1. Open `menu.html` in a browser.
2. Click `Add to Cart` on menu items.
3. Click the cart button to view your cart and checkout.
